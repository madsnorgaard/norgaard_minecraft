# Aurora's Kingdom Modpack Installation

## Download
Get the latest release from GitHub Releases

## Installation Steps
1. Install Fabric 1.21.5 first
2. Download modpack zip (110MB)
3. Extract the mods folder
4. Copy to .minecraft/mods/
5. Launch with Fabric profile

## Server Connection
Address: [your-server-ip]:25565

## Troubleshooting
- If mods show as corrupted: Delete and re-download
- Files should be 100KB+ minimum (not 1-2KB)
- Total: 90 working mods
