## Requirements
- Minecraft Java Edition (via Xbox Game Pass or purchased)
- CurseForge App

## Step 1: Install CurseForge
1. Download from: https://www.curseforge.com/download/app
2. Install and login with your account

## Step 2: Import Modpack
1. Download: `AurorasKingdom-FINAL.zip` from Dad
2. Open CurseForge
3. Click **"Create Custom Profile"** (+ button)
4. Select **"Import"**
5. Choose the downloaded zip file
6. Wait for import to complete

## Step 3: Launch Minecraft
1. In CurseForge, find "Aurora's Kingdom" profile
2. Click **Play**
3. This will open Minecraft with all mods loaded

## Step 4: Join Server
1. In Minecraft, click **Multiplayer**
2. Click **Add Server**
3. Enter:
   - Server Name: `Dad's Server`
   - Server Address: `theazanianprepper:25565`
4. Click **Done**
5. Double-click the server to join!

## Troubleshooting
- **Can't connect?** Make sure you launched from CurseForge
- **Wrong version?** The modpack uses Minecraft 1.21.5
- **Crashes?** Let CurseForge fully download all mods first
